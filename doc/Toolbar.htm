The Toolbar

The toolbar has icons for each of the items listed below.  Tooltip help will tell you what each one does.

Plot - Do it and see what it does.  This will plot a thousand or more points randomly about the screen (see the theory below for more info about that.

Stop - Stop a Plot that is in progress.  This is only enabled during a plot.

Options - This pulls up a dialog that you can use to control how the Random Game works.  See Options for details.  This button sticks down and you may close the Options dialog by clicking this button when the dialog is open.

Clear - Wipe out the plotting area with your current background color. You can select the background color from the Options dialog.

Set - This is only enabled if you clear the Create Regular Shape check box in the Options dialog.  This allows you to set the points for yourself, instead of letting the applet create then evenly spaced.  See the examples for some screen shots of some variations.

Histogram - This will display a histogram of the points plotted so you can see how random the random game really it. This window gets prettier when you plot twenty points instead of the default of three.  This button sticks down and you may close the histogram window by clicking this button when the dialog is open.

Print - Pulls up the print preview dialog from which you can print the current screen.

Help - That gets you here.
