The Options Dialog

Sides: The number or sides, or corners, this shape will have.  If you have Create Regular Shape checked, the game will plot a regular figure (all sides equal).  Otherwise, this it the number of points you can manually set using the Set button.

Points to plot:  This is the number of randomly generated points to plot each time the Plot button is pressed.  The more points you plot, the more filled in the image will be.

Divisor:  This number determines how far to go towards the next random point.  For a triangle, 2 and 0.667 are the most dramatic.  Since this is the divisor, 2 means 1/2 and 0.667 means 1/0.667 or 1.5.  You can play with this number to get different patterns.  See the examples for some screen shots that use different values for the divisor.

Use marker on set points:  If this is checked, each corner of the figure will be marked with 'x' marker and the randomly picked starting point for each plot will be marked with a asterisk-type marker.  This feature is useful when initially playing the game to see how the points are selected.

Create Regular Shape:  This disables the Set button and will set the corners of the figure before each plot.  

Regular Shape is checked.  This determines how big the regular figure should be.  This can be thought of as zoom.

Foreground and Background buttons:  As you can guess, these buttons pull up dialogs for setting the plotting colors.  The background color does not come into play unless you use the Clear toolbar button.  The new colors do not become effective until you press OK or Apply

OK and Apply buttons:  Commit any changes you've made in this dialog or in the color dialogs.  The Apply button leaves this dialog up so you can do a plot, adjust and plot without pulling it up each time.  The OK button commits the changes and closes the dialog. 

Cancel/Close button:  This will close the dialog.  You will lose any changes you've made since the last time you pressed Apply.
